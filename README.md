# 🚗 DriveFleet - Car Rental Platform

DriveFleet is a full-stack car rental platform where users can explore available cars, view car details, book vehicles, manage bookings, and add their own car listings. The platform includes secure authentication, protected routes, booking management, search & filtering functionality, and a modern responsive UI.

---

## 🌐 Live Website

🔗 Live Site: https://drivefleet-next-app.vercel.app/

---

## 📌 Features

- 🔐 Secure Authentication with Email/Password & Google Login
- 🚘 Add, Update, and Delete Car Listings
- 📅 Car Booking System with Booking Management
- 🔎 Search by Car Name & Filter by Car Type
- 🛡️ Protected Private Routes using JWT & Cookies
- 🌙 Responsive Modern UI with Dark/Light Theme
- ⚡ Loading Spinner & Custom 404 Not Found Page
- 📱 Fully Responsive for Mobile, Tablet, and Desktop

## 🔐 Environment Variables

Create a `.env.local` file in the root directory and add:

```env
NEXT_PUBLIC_API_URL=your_api_url
NEXT_PUBLIC_IMAGE_HOST_KEY=your_imgbb_key
BETTER_AUTH_SECRET=your_secret
BETTER_AUTH_URL=http://localhost:3000
```

## ⚙️ Installation & Setup

### Clone the repository

```bash
git clone https://github.com/your-username/drivefleet-client.git
```

### Navigate to the project

```bash
cd drivefleet-client
```

### Install dependencies

```bash
npm install
```

### Run the development server

```bash
npm run dev
```
